### Классификация жанров фильмов по описаниям (NLP)
**Технологии**: Python, PyTorch, SentenceTransformer, LSTM, Attention Mechanism

- Разработал модель классификации жанров фильмов по текстовым описаниям
- Применил BPE-токенизацию и предобученные embeddings (SentenceTransformer)
- Реализовал архитектуру LSTM с механизмом внимания для обработки последовательностей
- Обучил модель с учетом дисбаланса классов (class weighting)
- Достиг точности 72% на валидационной выборке
- Визуализировал результаты с помощью t-SNE для анализа качества embeddings
